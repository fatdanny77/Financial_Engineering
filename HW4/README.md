#### Risk neutral(風險中立)
風險中立的世界中假設投資人不會為了取得超額報酬承受額外的風險，他們願意只拿到無風險報酬。
由於過去的資訊已經反映在資產價格上，而未來的資產價格軌跡是未知的隨機過程，因此在一個風險中立的世界中未來的資產價格為其期望值，現值即為該期望值已無風險利率折現。
