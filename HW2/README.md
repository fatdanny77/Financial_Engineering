#### 有了 Spot Rate 以及 Forward Rate，我們便可畫出債券之殖利率曲線。正常來說，殖利率曲線應為正斜率，因為到期時間越長，風險越大，Duration 越高。然而，當殖利率曲線倒掛，為負斜率時，代表市場預期未來景氣不佳，是一個眾所注目的警訊。
